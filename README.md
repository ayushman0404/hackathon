# ⚡ NeuralDocs — AI PDF Intelligence Engine

Convert any PDF into structured JSON using AI.

## 🚀 Features

* Text extraction
* Image understanding (Gemini AI)
* Mixed content detection
* Page-wise summaries
* JSON export

## 🧠 Tech Stack

* Streamlit
* Gemini 2.5 Flash
* PyMuPDF + pdfplumber

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📌 Use Case

Hackathons, document AI tools, automation pipelines
